The Game

dnduet is a custom-built tabletop roleplaying system and the digital tools available below have been built with that system in mind. While the system needs little more than dice and something to write with, the tools are meant to help with creating characters, adventures, traps, puzzles, maps, items, perks, quirks, and more.
